## 📦 安装步骤

### 方式 1：安装版（推荐）

1. **下载文件**
   - 从 `dist-win7/` 目录复制 `QHD Invoice Generator Setup 2.2.0.exe` 到你的电脑

2. **运行安装程序**
   - 双击 `QHD Invoice Generator Setup 2.2.0.exe`
   - 如果出现"用户账户控制"提示，点击"是"

3. **安装向导**
   - 选择安装目录（默认：`C:\Program Files\QHD Invoice Generator`）
   - 勾选"创建桌面快捷方式"（推荐）
   - 点击"安装"

4. **完成安装**
   - 等待安装完成
   - 点击"完成"启动应用

### 方式 2：便携版

1. **下载文件**
   - 复制 `QHD Invoice Generator 2.2.0.exe` 到任意文件夹

2. **创建数据文件夹**（可选）
   - 在 exe 文件同目录创建 `data` 文件夹
   - 用于存储应用数据

3. **运行应用**
   - 双击 `QHD Invoice Generator 2.2.0.exe`

## ⚡ Windows 7 特别注意事项

### 1. 需要安装 Service Pack 1 (SP1)

Windows 7 **必须**安装 SP1 才能运行此应用。

**检查是否安装了 SP1：**
1. 点击"开始" → "计算机"右键 → "属性"
2. 查看"Windows 版本"部分
3. 应显示"Service Pack 1"

**如果没有 SP1：**
- 访问 Microsoft 官网下载并安装 Windows 7 SP1
- 或使用 Windows Update 自动安装

### 2. 可能需要安装 .NET Framework

某些 Windows 7 系统可能需要 .NET Framework 4.5 或更高版本。

**安装方法：**
1. 访问 Microsoft 官网
2. 下载 .NET Framework 4.5 或更高版本
3. 安装后重启电脑

### 3. Windows Defender / 杀毒软件警告

由于应用未经过代码签名，可能会被 Windows Defender 或其他杀毒软件标记。

**解决方法：**
1. 右键点击 exe 文件 → "属性"
2. 勾选"解除锁定"（如果有）
3. 点击"确定"
4. 或者在杀毒软件中添加例外

### 4. 显示问题

如果应用界面显示模糊或太小：

**调整 DPI 设置：**
1. 右键点击应用快捷方式 → "属性"
2. 切换到"兼容性"标签
3. 勾选"禁用显示缩放"
4. 点击"确定"

## 🐛 常见问题

### Q: 应用无法启动，提示缺少组件
**A:** 需要安装：
1. Windows 7 SP1
2. .NET Framework 4.5+
3. Visual C++ Redistributable 2015-2022

### Q: 安装时提示"Windows已保护你的电脑"
**A:** 点击"更多信息" → "仍要运行"

### Q: 应用运行很慢
**A:** 可能原因：
1. 电脑配置较低 - 升级内存到4GB+
2. 硬盘空间不足 - 清理磁盘
3. 后台程序太多 - 关闭不必要的程序

### Q: 能否在 Windows XP 上运行？
**A:** 不能。Windows XP 已不受支持。最低要求是 Windows 7 SP1。

### Q: 32位和64位有什么区别？
**A:**
- 64位版本可以使用更多内存，性能更好
- 32位版本兼容性更好，适合老旧电脑
- 此安装包自动选择合适的版本

### Q: 数据存储在哪里？
**A:**
- 安装版：`%APPDATA%\qhd-invoice-generator\`
- 便携版：exe 文件同目录下

### Q: 如何备份数据？
**A:** 复制上述数据目录即可

## 🔧 故障排除

### 应用无法启动

**步骤1：检查系统要求**
- Windows 7 SP1 ✓
- .NET Framework 4.5+ ✓
- 足够的磁盘空间 ✓

**步骤2：以管理员身份运行**
1. 右键点击应用
2. 选择"以管理员身份运行"

**步骤3：兼容性模式**
1. 右键点击应用 → "属性"
2. "兼容性"标签
3. 勾选"以兼容模式运行这个程序"
4. 选择"Windows 7"

**步骤4：重新安装**
1. 卸载现有版本
2. 重启电脑
3. 重新安装

### 打印问题

如果打印预览不显示：
1. 确保安装了打印机驱动
2. 尝试使用 PDF 打印机（如 Microsoft Print to PDF）
3. 检查浏览器打印设置
